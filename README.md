No installation required for static HTML/JavaScript. Simply open the `index.html` file in a web browser to view and interact with the application.

- To toggle theme: call `toggleTheme()`
- To add a product to cart: call `addToCart(productId)`
- To add a product to wishlist: call `addToWishlist(productId)`
